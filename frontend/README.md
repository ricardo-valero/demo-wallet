# React + TypeScript + Vite

Setup
```bash
pnpm create vite
```

Install dependencies

```bash
pnpm install
```

Run development server
```bash
pnpm vite
# or use the script in package.json
pnpm run dev
```


## Code quality

Setup
```bash
pnpm biome init
```

Lint and format
```bash
pnpm biome check --write <files>
# or use the script in package.json
pnpm run lint
```
